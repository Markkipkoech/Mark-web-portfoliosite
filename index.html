About Me

A bit about me

I'm a self-driven web developer based in Nakuru, Kenya. I got into coding out of genuine curiosity — I wanted to understand how websites worked, and one thing led to another.

On the frontend I work with HTML, CSS, and JavaScript to build interfaces that feel natural and load fast. On the backend I use Python — mostly Flask — to build APIs, handle databases, and wire everything together.

I've built a full-stack e-commerce project (Bora Poultry) which includes M-Pesa payment integration, user authentication, and an admin dashboard. It's one of the things I'm most proud of so far.

When I'm not building something, I'm usually reading about tech, exploring open-source repos, or experimenting with something new.
